# python_assignment_4
📄 File Handling Assignment – Read Text File Line by Line
📌 Overview

This project demonstrates how to read a text file line by line in Python and display each line with its corresponding line number. It also focuses on handling errors gracefully when the file is not present.

🎯 Objective
Read a text file (sample.txt)
Display content line by line with numbering
Handle missing file errors properly
🚀 Approaches Used
🔹 1. File Existence Check (os.path.exists())
Checks whether the file is available before opening it
Prevents runtime errors by handling the condition beforehand
Suitable for beginners and simple use cases
🔹 2. Exception Handling (try-except)
Attempts to open the file directly
Handles errors only if they occur
Considered a more Pythonic and efficient approach
✅ Features
Line-by-line file reading
Line numbering for better readability
Clean output formatting
Graceful error handling
Two different implementation approaches
⚠️ Error Handling

If the file is not found, the program displays an appropriate error message instead of crashing.

📚 Key Concepts Learned
File handling in Python
Reading files efficiently
Looping through file content
Error handling techniques
Difference between pre-check and exception handling
📌 Conclusion

This project provides a clear understanding of basic file handling in Python and demonstrates two effective ways to manage errors. It helps build a strong foundation for working with files in real-world applications.
