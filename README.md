# 📄 Assignment 4 – Files, Exceptions, and Errors in Python

## 📌 Overview

This assignment demonstrates file handling operations in Python, including reading from a file, writing data to a file, appending data, and handling errors using exception handling.

---

## 🎯 Task 1: Read a File and Handle Errors

### ✔ Description

* Open and read a file named `sample.txt`
* Display its content line by line
* Handle the error if the file does not exist

### ✔ Key Points

* File is opened in read mode
* Content is read line by line
* Each line is displayed clearly
* Error is handled using exception handling

---

## 🎯 Task 2: Write and Append Data to a File

### ✔ Description

* Take user input and write it to `output.txt`
* Append additional user input to the same file
* Read and display the final content of the file

### ✔ Key Points

* File is opened in write mode to store initial input
* File is opened in append mode to add more data
* Multiple lines of input are supported
* Final content is displayed after operations

---

## ⚠️ Error Handling

* Exception handling is used to prevent program crashes
* If a file is not found, an appropriate error message is displayed
* Ensures smooth execution of the program

---

## 📚 Concepts Used

* File handling (`read`, `write`, `append`)
* File modes (`r`, `w`, `a`)
* Loops for handling multiple lines
* Exception handling (`try-except`)
* User input handling

---

## 📌 Conclusion

This assignment helps in understanding how to work with files in Python and how to handle errors effectively. It provides practical knowledge of reading, writing, and updating files while ensuring the program runs safely.
